## Marcenaria Lajes

Landing page responsiva desenvolvida para a **Marcenaria Lajes**, empresa com 34 anos de experiência no ramo mobiliário localizada em São Paulo/SP.

## Sobre o Projeto

Site institucional com foco em apresentar os serviços, projetos realizados e informações de contato da marcenaria. O design prioriza elegância e sofisticação, com uma paleta de cores em preto e dourado.

### Funcionalidades

- **Slideshow** automático com imagens de projetos
- **Carrossel** interativo de portfólio (cozinhas e salas)
- **Seção de avaliações** de clientes
- **Mapa integrado** (Google Maps) com localização
- **Botão flutuante** do WhatsApp para contato direto
- **Menu responsivo** (hamburger menu) para dispositivos móveis
- **Design totalmente responsivo** (Desktop, Tablet e Mobile)

## Tecnologias Utilizadas

- **HTML5** — Estrutura semântica
- **CSS3** — Estilização com variáveis CSS (Custom Properties)
- **JavaScript** — Interatividade (slideshow, carrossel, menu mobile)
- **Font Awesome** — Ícones
- **Google Fonts** — Tipografia (Montserrat, Open Sans, Roboto)

## Responsividade

O site é responsivo com breakpoints em:
- **890px** — Layout tablet (menu hamburger, colunas empilhadas)
- **600px** — Layout mobile (carrossel single-slide, ajustes de fonte)
- **560px** — Layout mobile pequeno (fontes e espaçamentos reduzidos)
